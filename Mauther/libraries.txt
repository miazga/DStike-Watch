# Required Arduino Libraries for Mauther Firmware

## Installation via Arduino Library Manager

Open Arduino IDE:
Sketch → Include Library → Manage Libraries

## Required Libraries

1. U8g2
   - Author: oliver
   - Version: 2.34.x or higher
   - Description: Display library for OLED (SH1106)
   - Search: "U8g2"

2. VL53L0X
   - Author: Pololu
   - Version: 1.3.x or higher
   - Description: Time-of-Flight distance sensor
   - Search: "VL53L0X"

3. RTClib
   - Author: Adafruit
   - Version: 2.1.x or higher
   - Description: Real-Time Clock library (DS3231)
   - Search: "RTClib"

4. Adafruit NeoPixel ✨ **IMPORTANT!**
   - Author: Adafruit
   - Version: 1.11.x or higher
   - Description: NeoPixel/WS2812 RGB LED control
   - Search: "Adafruit NeoPixel"
   - **Required for RGB LED to work!**

## Built-in Libraries (No installation needed)

- Keyboard (ATmega32U4 HID)
- Wire (I2C communication)

## Verification

After installing, restart Arduino IDE and verify by checking:
Sketch → Include Library → (libraries should be listed)

## Alternative: Manual Installation

If Library Manager doesn't work:

1. U8g2:
   https://github.com/olikraus/u8g2
   Download ZIP → Sketch → Include Library → Add .ZIP Library

2. VL53L0X:
   https://github.com/pololu/vl53l0x-arduino
   Download ZIP → Sketch → Include Library → Add .ZIP Library

3. RTClib:
   https://github.com/adafruit/RTClib
   Download ZIP → Sketch → Include Library → Add .ZIP Library

## Troubleshooting

### Library Not Found Error
- Restart Arduino IDE after installation
- Check spelling of #include statements
- Verify library installed in correct folder

### Compilation Errors
- Update libraries to latest version
- Clear Arduino cache: Delete contents of temp folder
- Reinstall Arduino IDE if persistent

### Version Conflicts
- Use versions specified above
- Uninstall old versions first
- Check library dependencies

## Library Locations

**Windows**: 
`C:\Users\<username>\Documents\Arduino\libraries\`

**macOS**: 
`~/Documents/Arduino/libraries/`

**Linux**: 
`~/Arduino/libraries/`
